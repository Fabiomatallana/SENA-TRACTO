TUTORIAL COMO HACER CARDIO

es un ejercicio fisico, muscular y aerobico y este regularmente aporta muchos beneficios a la salud. Aumenta la energia y disminuye el porcentaje muscular.

1.CALENTAMIENTO

Como realizar el calentamiento haz ejercicios de intensidad baja durante un tiempo de 5 a 10 minutos para que se estimule el sistema cardio y aumentar el flujo sanguineo.ejemplo seria si es un gym utilizar la caminadora o caminar a un paso constante.

2. ACONDICIONAMIENTO

Es esencial para la preparacion del cuerpo para empezar con la actividad fisica en este ya se puede incluir trotar, correr andar en bisicleta o nadar tiene un tiempo estimado de 20 minutos.

3. ENFRIAMIENTO

Por ultimo y no menos importante despues de realizar cualquier actividad fisica es importante realizar un estiramineto de los musculos trabajados para evitar calambres y dolores muculares. siempre que se realiza una actividad fisca es importante el consumo de agua para la hidratación del cuerpo
